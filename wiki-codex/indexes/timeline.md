# Timeline

> This timeline tracks the field using publication dates from the corpus. For repos, the frontmatter dates are scrape dates, not launch dates, so the chronology below leans on papers, articles, and tweets where dated publication is explicit.

## 2023: Early Agent Loops And Reflective Memory

- `2023-03-20` — [Reflexion](../../raw/papers/shinn-reflexion-language-agents-with-verbal-reinforceme.md) frames verbal reinforcement as a practical loop for storing lessons outside model weights.
- `2023-05-25` — [Voyager](../../raw/papers/wang-voyager-an-open-ended-embodied-agent-with-large-l.md) shows open-ended skill acquisition in an embodied setting, making explicit that capability libraries can grow over time.
- `2023-06-23` — [Lil'Log on autonomous agents](../../raw/articles/lil-log-llm-powered-autonomous-agents.md) summarizes the early agent design space and establishes planning, memory, and tool use as the basic architecture questions.

## 2024: Retrieval Structure Gets More Serious

- `2024-04-24` — [GraphRAG](../../raw/papers/edge-from-local-to-global-a-graph-rag-approach-to-quer.md) pushes entity graphs and community summaries into the retrieval conversation.
- `2024-11-28` — [Lil'Log on reward hacking](../../raw/articles/lil-log-reward-hacking-in-reinforcement-learning.md) becomes relevant to agent self-improvement once more systems start optimizing themselves.

## 2025: Memory And Context Become Explicit Disciplines

- `2025-01-20` — [Zep](../../raw/papers/rasmussen-zep-a-temporal-knowledge-graph-architecture-for-a.md) makes temporal knowledge graphs a serious agent-memory architecture.
- `2025-02-17` — [A-MEM](../../raw/papers/xu-a-mem-agentic-memory-for-llm-agents.md) and [RAG vs. GraphRAG](../../raw/papers/han-rag-vs-graphrag-a-systematic-evaluation-and-key.md) sharpen the memory and retrieval debates around evolving context and graph structure.
- `2025-07-17` — [The context engineering survey](../../raw/papers/mei-a-survey-of-context-engineering-for-large-language.md) treats context engineering as a field rather than a loose collection of prompt tricks.
- `2025-09-19` — [Fabricio Q on episodic vs semantic memory](../../raw/articles/fabricio-q-memory-in-agents-episodic-vs-semantic-and-the-h.md) helps popularize hybrid layered memory as the practical default.
- `2025-10-06` — [ACE paper](../../raw/papers/zhang-agentic-context-engineering-evolving-contexts-for.md) reframes context as something that can evolve and accumulate learned strategy.
- `2025-10-09` — [Hugging Face on MemAgent](../../raw/articles/hugging-face-mem-agent-equipping-llm-agents-with-memory-using.md) highlights learned memory management with RL.

## Early 2026: Skills, Context Operating Systems, And Continuous Improvement

- `2026-02-12` — [Agent Skills for Large Language Models](../../raw/papers/xu-agent-skills-for-large-language-models-architectu.md) treats skills as a full research area spanning architecture, acquisition, and security.

- `2026-03-07` — [Karpathy packages Autoresearch](../../raw/tweets/karpathy-i-packaged-up-the-autoresearch-project-into-a-ne.md), making self-improvement loops legible to a much wider builder audience.
- `2026-03-16` — [The Agentic Researcher](../../raw/papers/zimmer-the-agentic-researcher-a-practical-guide-to-ai-as.md) shows how quickly research workflows are becoming agent-native.
- `2026-03-21` — [Autoresearch 101 Builder's Playbook](../../raw/articles/the-product-channel-by-sid-saladi-andrej-karpathy-s-autoresearch-101-builder-s-p.md) translates self-improvement ideas into a practical builder recipe.

## April 2026: The Wiki, Skills, And Context-Graph Explosion

- `2026-04-01` — [Akoratana’s context-graphs tweet](../../raw/tweets/akoratana-context-graphs-will-be-to-the-2030s-what-databases.md) captures the growing belief that context graphs may become foundational infrastructure.
- `2026-04-01` — [Omar Sar0’s universal `CLAUDE.md`](../../raw/tweets/omarsar0-universal-claude-md-claims-to-cut-claude-output-t.md) shows token discipline moving into mainstream coding-agent practice.
- `2026-04-02` — [Karpathy’s knowledge-base tweet](../../raw/tweets/karpathy-llm-knowledge-bases-something-i-m-finding-very-us.md) makes the markdown-compiled wiki pattern the dominant discourse object in this corpus.
- `2026-04-02` — [Pawel Huryn’s `CLAUDE.md` tweet](../../raw/tweets/pawelhuryn-your-claude-md-is-doing-jobs-that-rules-hooks-an.md) sharpens the distinction between rules, hooks, skills, and agents as separate context surfaces.
- `2026-04-03` — [Jumperz’s review-gated wiki variant](../../raw/tweets/jumperz-took-karpathy-s-wiki-pattern-and-wired-it-into-my.md) adds the missing validator between agent output and durable knowledge.
- `2026-04-03` — [OpenAI’s self-evolving agents cookbook](../../raw/articles/developers-openai-com-self-evolving-agents-a-cookbook-for-autonomous-a.md) formalizes eval-driven improvement loops for practitioners.
- `2026-04-04` — [Anthropic’s context engineering article](../../raw/articles/effective-context-engineering-for-ai-agents.md), [Martin Fowler’s coding-agent context memo](../../raw/articles/martinfowler-com-context-engineering-for-coding-agents.md), and [LangSmith’s trace loop article](../../raw/articles/langchain-com-the-agent-improvement-loop-starts-with-a-trace.md) together lock in the discipline-level framing: context, memory, and improvement are now systems design problems.

## What The Timeline Shows

The field did not progress in a straight line from RAG to memory to self-improvement. It progressively externalized more of the agent’s intelligence into inspectable artifacts: reflections, skills, graphs, context playbooks, traces, and finally compiled wikis. That is the through-line linking the older research papers to the 2026 builder wave.
