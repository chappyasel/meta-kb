# everything-claude-code

> A large agent harness that turns skills, hooks, memory, and context policies into a reusable operating environment. [Source](../../raw/repos/affaan-m-everything-claude-code.md)

## What It Does

everything-claude-code ships a broad workflow system for Claude Code and adjacent agents: skills, instincts, memory persistence, security layers, subagent orchestration, and compatibility support for several runtimes. [Source](../../raw/repos/affaan-m-everything-claude-code.md)

## Architecture

The repo is harness-first. Skills are the primary workflow surface, while hooks, contexts, MCP integration, and memory persistence manage session continuity and selective loading. The project explicitly frames itself as a performance-optimization system for agent harnesses. [Source](../../raw/repos/affaan-m-everything-claude-code.md)
