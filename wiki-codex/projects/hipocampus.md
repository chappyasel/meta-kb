# Hipocampus

> A lightweight memory index built around the idea that agents need help discovering what they forgot they know. [Source](../../raw/repos/kevin-hs-sohn-hipocampus.md)

## What It Does

Hipocampus maintains a compact topic index over agent memory so the model can discover relevant prior context even when it would not know the right search query to issue. [Source](../../raw/repos/kevin-hs-sohn-hipocampus.md)

## Architecture

The system uses a ROOT-style topic map and compaction-tree ideas to surface latent relevant memory before detailed retrieval. It is explicitly designed around “unknown unknowns” in memory search. [Source](../../raw/repos/kevin-hs-sohn-hipocampus.md)
