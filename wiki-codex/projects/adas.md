# ADAS

> A meta-agent search system for discovering better agent architectures by programming them directly. [Source](../../raw/repos/shengranhu-adas.md)

## What It Does

ADAS explores agent design automatically by having meta-agents generate and test new agentic architectures rather than assuming the scaffold is fixed. [Source](../../raw/repos/shengranhu-adas.md)

## Architecture

The project treats agent architecture as a search space and uses code generation to explore it, which places it closer to self-evolving systems than to ordinary prompt engineering. [Source](../../raw/repos/shengranhu-adas.md)

## Strengths

- Strong example of architecture-level improvement rather than prompt-level tweaking. [Source](../../raw/repos/shengranhu-adas.md)

## Limitations
