# Letta

> A stateful-agent platform that treats memory blocks as native agent state rather than optional retrieval garnish. [Source](../../raw/repos/letta-ai-letta.md)

## What It Does

Letta provides APIs and tools for building stateful agents with persistent memory blocks that survive across conversations and can support continual learning patterns. [Source](../../raw/repos/letta-ai-letta.md)

## Architecture

The core abstraction is the memory block: structured persistent state attached to the agent rather than replayed from old transcripts. The repo also exposes Letta Code skills and subagents for advanced workflows. [Source](../../raw/repos/letta-ai-letta.md)
