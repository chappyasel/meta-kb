# claude-skills

> A cross-platform skill registry that emphasizes portability across multiple coding agents rather than a single vendor surface. [Source](../../raw/repos/alirezarezvani-claude-skills.md)

## What It Does

`claude-skills` packages skills and agent plugins so the same capability can be reused across Claude Code, Gemini CLI, Codex, OpenClaw, and other toolchains. [Source](../../raw/repos/alirezarezvani-claude-skills.md)

## Architecture

The project standardizes skill packaging around `SKILL.md` plus supporting tooling, then adds installation and compatibility guidance for multiple agent runtimes. It treats skills as a portability layer. [Source](../../raw/repos/alirezarezvani-claude-skills.md)

## Strengths

- Strong multi-platform orientation. [Source](../../raw/repos/alirezarezvani-claude-skills.md)
- Useful when teams do not want one skill format per agent product. [Source](../../raw/repos/alirezarezvani-claude-skills.md)

## Limitations

- Breadth is the selling point, so the project is less opinionated about workflow quality than curated bundles. [Source](../../raw/repos/alirezarezvani-claude-skills.md)
