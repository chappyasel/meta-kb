# CORAL

> A multi-agent self-evolution framework where agents share attempts, notes, and skills through a public state layer. [Source](../../raw/repos/human-agent-society-coral.md)

## What It Does

CORAL provides lightweight infrastructure for autoresearch-style multi-agent work, with shared persistent knowledge in `.coral/public/` so agents can build on each other’s findings. [Source](../../raw/repos/human-agent-society-coral.md)

## Architecture

The repo emphasizes public shared state for attempts, notes, and skills rather than siloed agents, making knowledge reuse a system feature rather than an accident. [Source](../../raw/repos/human-agent-society-coral.md)
