# MemoryBank-SiliconFriend

> A memory system built around selective reinforcement and forgetting rather than naive accumulation. [Source](../../raw/repos/zhongwanjun-memorybank-siliconfriend.md)

## What It Does

MemoryBank uses significance and recency to strengthen or decay memories over time, aiming to keep long-horizon conversational agents coherent without retaining every old detail equally. [Source](../../raw/repos/zhongwanjun-memorybank-siliconfriend.md)

## Architecture

The design is inspired by Ebbinghaus-style forgetting curves, treating memory retention as a dynamic process instead of a pure storage problem. [Source](../../raw/repos/zhongwanjun-memorybank-siliconfriend.md)
