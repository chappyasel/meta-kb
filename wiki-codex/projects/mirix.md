# MIRIX

> A multi-agent memory system that routes different memory types to different specialized stores. [Source](../../raw/repos/mirix-ai-mirix.md)

## What It Does

MIRIX organizes memory into six specialized agents or stores, including episodic, semantic, procedural, resource, and vault-like components, with screen-grounded support for personal-assistant workflows. [Source](../../raw/repos/mirix-ai-mirix.md)

## Architecture

Its key bet is specialization by memory type. Instead of one flat memory store, the system routes different kinds of context to different memory roles that mirror human cognitive distinctions. [Source](../../raw/repos/mirix-ai-mirix.md)
