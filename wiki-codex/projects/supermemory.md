# Supermemory

> A unified memory engine that emphasizes temporal reasoning, contradiction handling, and automatic fact extraction. [Source](../../raw/repos/supermemoryai-supermemory.md)

## What It Does

Supermemory stores and recalls agent memory through a unified ontology that can ingest information, extract facts, and keep track of temporal and contradictory state. [Source](../../raw/repos/supermemoryai-supermemory.md)

## Architecture

The repo positions the system as a memory engine rather than a bare vector database, with plugins, MCP integration, and a “give your AI memory” product framing. [Source](../../raw/repos/supermemoryai-supermemory.md)

## Key Numbers

- 20,994 GitHub stars. [Source](../../raw/repos/supermemoryai-supermemory.md)

## Strengths

- Strong focus on problems many memory systems underplay: contradictions, temporal drift, and forgetting. [Source](../../raw/repos/supermemoryai-supermemory.md)

## Limitations

- Public repo materials are more product-facing than benchmark-heavy in the excerpted corpus. [Source](../../raw/repos/supermemoryai-supermemory.md)
