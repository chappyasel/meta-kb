# Obsidian Skills

> A skills library that brings agentic automation directly into an Obsidian-based knowledge environment. [Source](../../raw/repos/kepano-obsidian-skills.md)

## What It Does

Obsidian Skills packages workflows for markdown creation, canvas use, CLI interaction, and vault manipulation so agents can operate inside Obsidian as a live knowledge environment. [Source](../../raw/repos/kepano-obsidian-skills.md)

## Architecture

The project treats Obsidian as an agent-facing workspace and distributes modular skills around that surface, including marketplace and multi-agent installation paths. [Source](../../raw/repos/kepano-obsidian-skills.md)

## Key Numbers

- 19,325 GitHub stars. [Source](../../raw/repos/kepano-obsidian-skills.md)

## Strengths

- Strong fit for markdown-first personal knowledge systems. [Source](../../raw/repos/kepano-obsidian-skills.md)
- Connects the skills abstraction directly to a real end-user vault workflow. [Source](../../raw/repos/kepano-obsidian-skills.md)

## Limitations

- Best when Obsidian is already the frontend; it is not a general-purpose agent platform by itself. [Source](../../raw/repos/kepano-obsidian-skills.md)

## Alternatives

- [Anthropic Skills](anthropic-skills.md) for a tool-agnostic skill standard.
- [Ars Contexta](ars-contexta.md) for deriving the vault structure itself.
