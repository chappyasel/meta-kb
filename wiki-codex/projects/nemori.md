# Nemori

> An episodic-memory system that treats event segmentation as the key design variable rather than just storage backend choice. [Source](../../raw/repos/nemori-ai-nemori.md)

## What It Does

Nemori builds conversational memory around event boundaries identified by the model, aiming to segment experiences more like human episodic memory and less like arbitrary fixed windows. [Source](../../raw/repos/nemori-ai-nemori.md)

## Architecture

The system focuses on event segmentation, episodic storage, and semantic distillation from those episodes rather than dumping all interactions into one memory stream. [Source](../../raw/repos/nemori-ai-nemori.md)
