## Tweet by @coreyganim

How to make your OpenClaw agent learn from its mistakes:

Install the Self-Improving Agent skill.

What it does:
→ Logs every error to .learnings/ERRORS.md
→ Captures corrections you make to .learnings/LEARNINGS.md
→ Tracks feature requests to .learnings/FEATURE_REQUESTS.md

How it improves over time:
→ Agent reviews past learnings before major tasks
→ Recurring issues get flagged and prioritized
→ Important lessons get promoted to AGENTS.md (permanent memory)

Setup takes 60 seconds:

1. Create .learnings/ folder in your workspace
2. Add the skill to your OpenClaw agent
3. That's it. Agent now logs mistakes automatically.

Day 1: Your agent makes the same errors everyone's does.
Day 30: Your agent knows YOUR business, YOUR tools, YOUR preferences.
