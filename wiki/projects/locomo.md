# LoCoMo

> A benchmark and dataset for evaluating long-context conversational memory across extended interactions. [Supermemory source](../../raw/repos/supermemoryai-supermemory.md)

## Why It Matters

LoCoMo matters because many memory systems in this field use it as shared evidence, which makes it one of the benchmark anchors for comparing long-horizon conversational recall. [Supermemory source](../../raw/repos/supermemoryai-supermemory.md)
