# MemAgent

> A memory-augmented agent system that uses reinforcement learning to optimize memory read and write operations for long-context tasks. [MemAgent source](../../raw/repos/bytedtsinghua-sia-memagent.md)

## Why It Matters

MemAgent is one of the clearer examples in this corpus of memory policy being learned rather than hand-designed, which pushes agent memory closer to continual adaptation. [MemAgent source](../../raw/repos/bytedtsinghua-sia-memagent.md)
