# Skill Composition

> Skill composition is the practice of combining multiple agent skills into a larger workflow so capabilities remain modular and reusable. [Agent Skills overview](../../raw/articles/agent-skills-overview.md)

## Why It Matters

It is one of the main ways agent systems avoid monolithic prompts: tasks are decomposed into reusable capability units that can be mixed, loaded, and chained. [Anthropic Skills source](../../raw/repos/anthropics-skills.md) [gstack source](../../raw/repos/garrytan-gstack.md)
