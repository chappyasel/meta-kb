# Multimodal Memory

> Multimodal memory stores and retrieves information across more than one modality, including text, images, OCR outputs, audio, or video-derived artifacts. [SimpleMem source](../../raw/repos/aiming-lab-simplemem.md)

## Why It Matters

It pushes agent memory beyond chat and code, which is necessary if knowledge systems are going to handle real-world files and media instead of only text transcripts. [Supermemory source](../../raw/repos/supermemoryai-supermemory.md)
