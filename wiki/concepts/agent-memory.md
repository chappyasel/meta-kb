# Agent Memory

> Agent memory is the set of mechanisms by which an agent stores, retrieves, updates, and forgets information across turns and sessions. [A-MEM](../../raw/papers/xu-a-mem-agentic-memory-for-llm-agents.md)

## Why It Matters

Without memory, agents reset too often; with bad memory, they accumulate stale or noisy state. That is why the field keeps splitting memory into types, scopes, and retention policies. [Mem0 source](../../raw/repos/mem0ai-mem0.md) [Graphiti source](../../raw/repos/getzep-graphiti.md)
