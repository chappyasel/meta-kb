# Self-Improving Agent

> A self-improving agent changes its own behavior, memory, or skill set in response to experience instead of relying only on static prompting. [OpenAI self-evolving agents cookbook](../../raw/articles/developers-openai-com-self-evolving-agents-a-cookbook-for-autonomous-a.md)

## Why It Matters

This is the core aspiration behind autoresearch loops, trace-driven repair, and evolving memory systems: behavior should get better after repeated use, not just differ. [Autoresearch source](../../raw/repos/karpathy-autoresearch.md) [Memento source](../../raw/repos/agent-on-the-fly-memento.md)
