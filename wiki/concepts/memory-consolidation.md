# Memory Consolidation

> Memory consolidation is the process of organizing, compressing, and moving information from short-term working state into longer-term memory. [Anthropic on context engineering](../../raw/articles/effective-context-engineering-for-ai-agents.md)

## Why It Matters

It is the hidden mechanism behind compaction, session summarization, and many forms of long-term agent memory. [Claude-Mem source](../../raw/repos/thedotmack-claude-mem.md) [Supermemory source](../../raw/repos/supermemoryai-supermemory.md)
