# Timeline

This timeline is based on the captured source dates in `raw/`. It is a timeline of the current corpus, not a full history of the field.

- 2025-01-20: The Zep paper formalizes temporal knowledge graphs for agent memory and reports enterprise-style benchmark gains, making provenance-heavy memory a serious production story. [Zep paper](../../raw/papers/rasmussen-zep-a-temporal-knowledge-graph-architecture-for-a.md)
- 2025-02-17: The GraphRAG evaluation paper argues that graph structure wins on some tasks and loses on others, pushing the field toward hybrid retrieval rather than one universal answer. [RAG vs. GraphRAG](../../raw/papers/han-rag-vs-graphrag-a-systematic-evaluation-and-key.md)
- 2025-02-17: A-MEM proposes dynamic, Zettelkasten-like memory organization where new notes can update older ones instead of only being appended. [A-MEM](../../raw/papers/xu-a-mem-agentic-memory-for-llm-agents.md)
- 2026-02-12: The agent-skills survey frames the skill layer as a real architecture category and highlights registry governance and security as first-class problems. [Agent Skills survey](../../raw/papers/xu-agent-skills-for-large-language-models-architectu.md)
- 2026-03-07: Karpathy's autoresearch tweet pushes the metric-first overnight loop into the mainstream with 28,330 likes and 10.9M views. [Karpathy autoresearch tweet](../../raw/tweets/karpathy-i-packaged-up-the-autoresearch-project-into-a-ne.md)
- 2026-03-31: The captured Memento repo snapshot shows case-based continual learning arriving as a practical open-source pattern without model-weight updates. [Memento source](../../raw/repos/agent-on-the-fly-memento.md)
- 2026-04-02: Karpathy's knowledge-base tweet turns raw-to-wiki markdown compilation into the signature pattern for inspectable agent knowledge systems. [Karpathy on LLM knowledge bases](../../raw/tweets/karpathy-llm-knowledge-bases-something-i-m-finding-very-us.md)
- 2026-04-03: The Agent Skills overview lands as a mainstream explanation of skills as portable capability packages. [Agent Skills overview](../../raw/articles/agent-skills-overview.md)
- 2026-04-03: The captured snapshots for [Mem0](../projects/mem0.md), [Letta](../projects/letta.md), [Supermemory](../projects/supermemory.md), and [GOAL.md](../projects/goal-md.md) show memory, stateful agents, managed context engines, and constructed metrics all maturing at the same time. [Mem0 source](../../raw/repos/mem0ai-mem0.md) [Letta source](../../raw/repos/letta-ai-letta.md) [Supermemory source](../../raw/repos/supermemoryai-supermemory.md) [goal-md source](../../raw/repos/jmilinovich-goal-md.md)
- 2026-04-04: Anthropic publishes "Effective context engineering for AI agents," making attention-budget language the dominant framing for prompt and memory design. [Anthropic on context engineering](../../raw/articles/effective-context-engineering-for-ai-agents.md)
- 2026-04-04: Martin Fowler publishes a coding-agent memo on rules, skills, control planes, and context loading, showing the ideas have crossed into mainstream software architecture discourse. [Martin Fowler on coding-agent context engineering](../../raw/articles/martinfowler-com-context-engineering-for-coding-agents.md)
- 2026-04-04: Large harness and skill repos like [Anthropic Skills](../projects/anthropic-skills.md), [Everything Claude Code](../projects/everything-claude-code.md), and [gstack](../projects/gstack.md) are all at scale, which is why "agent systems" now looks like an ecosystem and not a collection of hacks. [Anthropic Skills source](../../raw/repos/anthropics-skills.md) [Everything Claude Code source](../../raw/repos/affaan-m-everything-claude-code.md) [gstack source](../../raw/repos/garrytan-gstack.md)
